dhtmlxTreeGrid

You can show nested tabular data using the special extension for dhtmlxGrid - dhtmlxTreeGrid. Additional .js file that should be added to the project is dhtmlxtreegrid.js. See dhtmlxGrid manual to get details about the grid structure initialization and usage.
